# JSO Career Portfolio Intelligence Agent

An AI-powered career analysis tool built for JSO Phase-2 Agentic Assignment.

## Features
- Upload Resume PDF
- Analyze Skill Gaps
- Career Recommendations
- Weekly Job Listing Updates
- LinkedIn & GitHub Profile Analysis

## Tech Stack
- HTML, CSS, JavaScript
- Google Gemini API / Groq API
- PDF text extraction

## How to Use
1. Open career_portfolio_agent.html in browser
2. Enter your API key
3. Upload your Resume PDF
4. Click "Run Intelligence Analysis"
